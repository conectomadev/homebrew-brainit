# homebrew-brainit

Homebrew tap for the **brainit** CLI — turn a local repo into code-embeddings
docs + semantic vectors, entirely on your machine.

```sh
brew install conectomadev/brainit/brainit
brainit --yes
```

Or without Homebrew:

```sh
curl -fsSL https://www.brainit.dev/install.sh | bash
```
